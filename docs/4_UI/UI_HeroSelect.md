# UI_HeroSelect - 初始英雄选择界面

> [Demo] 3名初始英雄可选（艾拉/西格玛/赛拉），不做解锁条件、难度选择、确认阶段。

## 概述

玩家开始新一局时，从已解锁的初始英雄中选择1名作为起始角色。选择后确认队伍配置，进入恶魔塔。

### 卡牌交互
- **悬停**：卡牌微微放大（110%），显示角色立绘动态（呼吸动画）
- **选中**：卡牌高亮金色边框，底部详情面板更新
- **未解锁**：显示剪影+锁定图标，悬停提示解锁条件
- **双击/再次点击**：直接确认选择并开始

### 属性评级显示
使用S/A/B/C/D/E六级评价，直观展示角色能力倾向：
- S（金色）= 超群
- A（红色）= 优秀
- B（蓝色）= 良好
- C（绿色）= 普通
- D（灰色）= 较差
- E（白色）= 极差
