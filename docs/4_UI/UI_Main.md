# UI_Main - 主界面与标题设计

> [Demo] 仅做标题+新游戏按钮+退出。不做存档管理、设置界面、图鉴。

## 概述

主界面是玩家进入游戏后的第一印象，需要营造"中世纪黑暗奇幻"氛围，同时提供清晰的导航入口。

### 交互逻辑
- **新游戏**：直接进入 HeroSelect 界面。若已有进行中的存档，弹窗确认"是否放弃当前进度？"
- **继续游戏**：检测存档文件，无存档时置灰不可点击
- **图鉴收集**：展示已解锁的英雄、敌人、装备图鉴
- **设置**：音量、语言、画面质量、键位配置
- **背景**：恶魔塔剪影缓慢呼吸式缩放，粒子效果为飘浮的灰烬

### 状态说明
| 状态 | 触发条件 |
|------|----------|
| 正常 | 无存档或已通关 |
| 有存档 | 检测到中途存档，"继续游戏"高亮 |
| 已通关 | 显示通关记录（最高难度、胜场等） |

```
┌──────────────────────────────────────┐
│  [画面] [音频] [游戏] [操作]         │
├──────────────────────────────────────┤
│                                      │

```

- **画面**：分辨率、全屏、帧率、抗锯齿、阴影、亮度
- **音频**：主音量、BGM音量、SFX音量、语音音量
- **游戏**：语言、伤害显示速度、镜头移动速度、网格显示开关、确认提示开关
- **操作**：键鼠绑定 / 手柄映射、镜头灵敏度

## 转场动画

| 转场 | 动画效果 | 时长 |
|------|----------|------|
| 标题→新游戏 | 黑幕渐隐，塔门开启转场 | 1.5s |
| 标题→继续 | 快速黑幕切换 | 0.5s |
| 军营→进塔 | 传送阵光效 | 1.0s |
| 战后→奖励 | 胜利特写→卡片翻转展开 | 1.0s |
| 死亡→标题 | 灰色滤镜覆盖→黑幕 | 2.0s |
| 通关→标题 | 光芒扩散→白幕→淡入 | 3.0s |
