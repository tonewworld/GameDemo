# UI_Camp - 营地/商店/事件界面

> [Demo] 实现商店(3商品+离开)和篝火(休整+离开)。事件界面复刻Module_Event中3个模板。

## 概述

非战斗节点UI包括三种类型：篝火营地（恢复与选择）、商店（货币交易）、事件（叙事+选择）。这些节点的UI需要营造"喘息感"——区别于战斗界面的紧张氛围。


### 商品刷新规则
- 基础商品：3件随机装备/消耗品
- 特卖品：1件稀有物品（每商店固定1件）
- 商店不可刷新（除非拥有特定祝福）
- 出售价格为购买价的50%

### 使魔商人特殊交互
- 使魔商人有随机性格（慷慨/狡猾/疯狂）
- 慷慨：全场9折
- 狡猾：随机1件商品标价翻倍但品质也翻倍
- 疯狂：出售诅咒物品，效果极强但有代价

### 布局
```
┌──────────────────────────────────────────┐
│                                          │
│        [事件插画/情境图]                   │
│                                          │
│  ┌────────────────────────────────────┐  │
│  │                                    │  │
│  │  "前方的走廊传来微弱的哭声。         │  │
│  │   一个平民被恶魔困在坍塌的石柱下，   │  │
│  │   正在绝望地呼救..."               │  │
│  │                                    │  │
│  │  — 救援平民                        │  │
│  │                                    │  │
│  └────────────────────────────────────┘  │
│                                          │
│  ┌────────────────────────────────┐      │
│  │ 选择你的行动：                  │      │
│  │                                │      │
│  │ ▶ [力量] 搬开石柱（STR检定）   │      │
│  │    需要STR ≥ 12               │      │
│  │    成功：平民获救+装备奖励      │      │
│  │    失败：石柱坍塌，HP-15       │      │
│  │                                │      │
│  │ ▶ [支付] 100灵魂碎片请求使魔帮助│      │
│  │    必定成功，获得使魔好感        │      │
│  │                                │      │
│  │ ▶ [离开] 无视哭声继续前进       │      │
│  │    无事发生                    │      │
│  │                                │      │
│  └────────────────────────────────┘      │
│                                          │
│  队伍STR最高：莉莉丝(STR:16) ✓ 满足       │
└──────────────────────────────────────────┘
```

### 事件类型分类
| 类型 | 描述 | 示例 |
|------|------|------|
| 救援 | 拯救平民/友军 | 搬开石柱、治疗伤员 |
| 狩猎 | 追猎隐藏怪物 | 追踪血迹、设伏突袭 |
| 角色挑战 | 特定角色的个人任务 | 莉莉丝"击败3个恶魔" |
| 谜题 | 机关解谜 | 压力板、推箱子 |
| 赌博 | 风险与回报 | 献祭HP换取随机装备 |
| 叙事 | 纯故事推进 | 发现前人笔记、记忆碎片 |
| 商人 | 特殊交易 | 诅咒物品交换、赌博商人 |

### 属性检定显示
- 检定选项前显示对应属性图标
- 显示所需值 vs 队伍最高值
- 满足条件时显示绿色✓，不满足时显示红色✗
- 不满足条件的选项仍可选择，但提示失败风险
