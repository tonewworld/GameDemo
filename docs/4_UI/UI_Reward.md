# UI_Reward - 战后奖励选择界面

> [Demo] 三选一：新英雄/装备/消耗品。不做祝福诅咒、稀有度光效、跳过机制、技能升级。

## 概述

每场战斗胜利后，玩家获得**"命运三选一"**——从3个随机奖励中选择1个。这是肉鸽构建的核心节点，设计参考《杀戮尖塔》的卡牌奖励。

---

## 奖励类型

| 类型 | 出现权重 | 稀有度分布 |
|------|----------|------------|
| 新英雄加入 | 40% | 普通70%/稀有25%/传说5% |
| 装备/圣物 | 35% | 普通60%/稀有30%/传说10% |
| 祝福/诅咒 | 15% | 祝福70%/诅咒30% |
| 灵魂碎片（额外） | 10% | 100/200/500 |

Boss战奖励池独立，保底包含1个传说品质。

### 卡牌交互
- 每张卡牌悬停时上浮+放大，显示完整属性对比
- 英雄卡牌悬停时显示与当前队伍羁绊关系预览
- 装备卡牌悬停时显示与已有装备的协同/冲突
- 祝福卡牌悬停时显示持续时间说明

## 奖励池设计

### 英雄奖励池
- 每位英雄有"出现权重"，受当前楼层、队伍构成、已解锁英雄影响
- 已拥有英雄的"复制体"出现概率降低70%
- 第1-5层保底1次英雄奖励

### 装备奖励池
- 按装备槽位分类：武器/防具/饰品
- 智能检测：队伍中无人可使用剑时，剑类武器出现概率降低
- 传说品质仅在精英/Boss/事件中出现

### 祝福/诅咒池
- 祝福：持续本层或固定回合数的正向Buff
- 诅咒：给予负面效果但提供额外灵魂碎片补偿
- 特殊"双刃"祝福：强正面+弱负面组合

---

## 跳过机制

玩家可以选择"跳过"奖励，获得保底的50灵魂碎片。适用于：
- 三个选项都不适合当前构建
- 不想扩大队伍规模（已经满员/不想稀释经验）
- 已有同类装备/祝福
