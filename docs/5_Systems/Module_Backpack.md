# Module_Backpack - 背包系统（Demo版）

> [Demo] 固定6格容量，仅存消耗品。装备直接穿戴不入包。

---

## Demo 背包规则

| 参数 | 值 |
|------|-----|
| 容量 | 6格 |
| 同物品堆叠上限 | 3 |
| 可存放 | 消耗品、投掷物 |
| 装备 | 直接穿戴，不入包 |
| 出售 | 商店可半价出售 |

---
