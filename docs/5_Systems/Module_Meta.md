# Module_Meta - 局外Meta成长系统

> **[Demo] 整个模块不做。Demo 每次开局固定初始配置，通关/死亡直接返回标题。此文档保留供后续参考。**

## 概述

Meta系统提供跨局的永久成长，让玩家在每次失败后仍有获得感。核心货币是**"灵魂残片"**——无论通关还是死亡都会保留。

---

## 一、Meta等级

### 概念
Meta等级独立于局内角色等级，随灵魂残片累积自动提升。每次升级解锁新能力/物品/英雄。

### 经验曲线
| Meta等级 | 所需灵魂残片 | 累计 |
|----------|-------------|------|
| Lv.1 → 2 | 200 | 200 |
| Lv.2 → 3 | 400 | 600 |
| Lv.3 → 4 | 700 | 1,300 |
| Lv.4 → 5 | 1,100 | 2,400 |
| Lv.5 → 6 | 1,600 | 4,000 |
| ... | +500/Lv | — |
| Lv.Max | — | — |

## 四、成就与图鉴

### 成就系统
成就独立于Meta等级，提供一次性灵魂碎片奖励。

| 成就 | 条件 | 奖励 |
|------|------|------|
| 初次攀登 | 到达第10层 | 300 |
| 恶魔杀手 | 累计击杀100只恶魔 | 500 |
| 完美战术 | 一场战斗全员无伤 | 200 |
| 精英猎手 | 累计击杀20个精英敌人 | 500 |
| 羁绊大师 | 触发所有羁绊连携技 | 800 |
| 独行侠 | 仅用1名英雄通关第1章 | 1,000 |
| 不败传说 | 通关噩梦难度 | 3,000 |

### 恶魔图鉴
击杀某种敌人累计一定数量后解锁图鉴条目，查看详细属性与掉落。

### 记忆碎片收集
特定事件/Boss/层数可找到"记忆碎片"，组合解锁隐藏剧情。
