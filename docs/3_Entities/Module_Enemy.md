## 二、Boss

### 守门者·加鲁姆（第5层）
| 属性 | 值 |
|------|-----|
| HP | 300 |
| STR | 18 |
| DEF | 16 |
| MOV | 3 |

**技能**：
- **横扫**：前方3格扇形 AOE，STR×1.2
- **咆哮**：自身周围2格内友方DEF-5（2回合）
- **狂暴**（HP<50%）：每回合行动2次

### 虚无骑士（第10层）
| 属性 | 值 |
|------|-----|
| HP | 500 |
| STR/MAG | 20/20 |
| DEF/RES | 18/18 |
| MOV | 4 |

**技能**：
- **暗影斩**：直线4格，STR×1.5
- **虚空弹**：远程5格单体，MAG×1.8
- **暗黑光环**（HP<50%）：全场友方-10%属性

> [Demo] Boss不做阶段转换、部位破坏、场地机制。

## 三、Demo AI（简化行为树）

```
AI.TakeTurn():
  1. 遍历技能，找"可击杀"目标 → 使用技能
  2. 遍历技能，找"AOE覆盖≥2目标" → 使用AOE
  3. 选择"最近且可攻击"的目标 → 移动+基础攻击
  4. 无目标在攻击范围 → 向最近敌人移动
```

> [Demo] 不做生存判断（撤退/治疗），不做地形利用。这是最简AI，足够展示战棋玩法。
