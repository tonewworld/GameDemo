## 二、六维属性（Demo 范围）

| 属性 | 说明 | Lv1 范围 |
|------|------|----------|
| HP | 生命值 | 80~130 |
| STR | 物理攻击 | 8~18 |
| MAG | 魔法攻击 | 6~20 |
| SPD | 速度（闪避+敌方回合顺序） | 6~16 |
| DEF | 物理防御 | 6~18 |
| RES | 魔法防御 | 6~18 |
| MOV | 每回合移动格数 | 3~6 |

## 三、技能系统（Demo简化）

每个英雄 **2个主动技能 + 1个被动**，升级时获得属性提升。

| 英雄 | 主动1 | 主动2 | 被动 |
|------|-------|-------|------|
| 艾拉 | 圣光斩（直线3格） | 铁壁（自身DEF+50%，2回合） | 骑士誓言：相邻友方DEF+3 |
| 西格玛 | 火药射击（3×3范围） | 炸药桶（放置可爆炸障碍） | 爆破专家：对障碍物伤害×2 |
| 赛拉 | 穿云箭（直线5格） | 箭雨（3×3范围） | 鹰眼：高地命中+15% |
| 马库斯 | 旋风斩（周围8格） | 挑衅（强制目标攻击自己） | 反击强化：反击伤害+30% |
| 莉莉丝 | 暗影弹（单体+腐蚀） | 恐惧术（目标跳过下次行动） | 腐蚀反噬：受击时反弹MAG×0.3伤害 |

> [Demo] 不做技能升级三选一。英雄升级时自动获得 STR/MAG+2 或 DEF/RES+2。
> [Demo] 不做羁绊系统、不做转职系统。留到正式版。
