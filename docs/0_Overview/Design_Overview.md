# Design_Overview - 设计总览（Demo版）

> **[Demo] 此文档描述完整设计愿景，Demo 只实现标 ✅ 的部分。**

---

## 世界观（简短版）

异世界 **艾尔德拉**，人类在活体建筑**"恶魔塔"**的阴影下生存。塔内空间每次进入自动重构。人类研发出**"弈者契约"**，将战斗具象化为战棋规则以求生存。

> [Demo] 世界观用 2-3 句开场文字交代即可，不需要过场动画。

---

## 核心玩法循环 ✅

```
选初始英雄 → 爬塔路线选择 → 战棋战斗 → 三选一奖励 → 循环至通关/死亡
```

| 步骤 | Demo 实现 |
|------|-----------|
| 选初始英雄 | 3选1，展示属性对比 |
| 路线选择 | 每层2-3条分支（战斗/商店/篝火/事件） |
| 战棋战斗 | 我方回合→敌方回合，网格移动+攻击 |
| 三选一奖励 | 新英雄/装备/消耗品随机 |

## Demo 塔层结构 ✅

共 **10层**，单章节（迷失回廊主题）：

| 层数 | 固定内容 |
|------|----------|
| 第1层 | 普通战斗（教程级，1个敌人） |
| 第2-4层 | 随机节点（战斗/商店/事件/篝火） |
| 第5层 | 中Boss（守门者·加鲁姆） |
| 第6-9层 | 随机节点，敌人强度上升 |
| 第10层 | 最终Boss（虚无骑士） |

## 视觉风格（Demo 简化）

- **视角**：2.5D 斜俯视 45°，正方形网格 ✅
- **场景**：石板地面 + 石柱 + 火炬光源，用免费 Asset Store 素材拼 ✅
- **角色**：Cube + 不同颜色材质区分（灰盒阶段），后续替换模型
- **UI**：深色底 + 金色边框，Unity UI（Canvas + TextMeshPro）

> [Demo] Demo 不必追求最终美术。用**色块/几何体**区分角色类型即可：蓝色=友方、红色=敌方、黄色=中立。面试官看的是系统和代码，不是美术。
