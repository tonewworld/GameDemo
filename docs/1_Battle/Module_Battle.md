# Module_Battle - 战棋战斗系统

> [Demo] 实现网格移动+攻击+回合交替+伤害公式+技能系统+5种状态。跳过双击判定、地形联动、复杂AI。

## 概述

战斗系统是整个游戏的核心，融合传统战棋的深度策略与肉鸽的随机性。采用经典的**我方回合→敌方回合**交替制（参考《火焰纹章》），SPD属性决定同方单位在回合内的行动顺序，但不改变回合交替的基本结构。

## 一、棋盘与网格

### 基础规则
- **网格类型**：正方形网格（Square Grid）
- **单层尺寸**：12×12 ~ 20×20（随楼层增加而扩大）
- **高度层级**：支持3层高度差（-1/0/+1），影响射程、伤害、移动消耗
- **移动消耗**：平地1MOV/格，上坡1.5MOV/格，水域2MOV/格

### 格子属性
| 属性 | 说明 |
|------|------|
| 地形类型 | 平地/水域/熔岩/毒沼/冰面/废墟/草丛 |
| 高度 | -1(低地)/0(平地)/+1(高地) |
| 可通行 | 飞行单位无视地形限制 |
| 可破坏 | 部分地形可被技能破坏（炸药桶、石柱） |
| 掩体值 | 提供防御加成，被攻击方向决定是否生效 |

### 攻击朝向加成
| 攻击方向 | 命中修正 | 暴击修正 | 说明 |
|----------|----------|----------|------|
| 正面 | ±0% | ±0% | 基准 |
| 侧面 | +10% | +15% | 需要攻击者位于目标侧方90° |
| 背面 | +20% | +30% | 需要攻击者位于目标后方90° |

## 二、回合系统

### 核心机制：我方回合 → 敌方回合交替制

每个回合分为两个阶段：

```
┌─────────────────────────────────────────────┐
│              第 N 回合                       │
│                                             │
│  ┌─────────────────────────────────────┐    │
│  │       我方回合（Player Phase）        │    │
│  │  · 可自由选择行动顺序                 │    │
│  │  · 每个单位：移动 + 主行动            │    │
│  │  · 全部行动完毕或手动结束回合          │    │
│  └─────────────────────────────────────┘    │
│                    ↓                        │
│  ┌─────────────────────────────────────┐    │
│  │       敌方回合（Enemy Phase）         │    │
│  │  · AI按SPD顺序依次行动                │    │
│  │  · 每个单位：移动 + 主行动            │    │
│  │  · 全部行动完毕后自动切换             │    │
│  └─────────────────────────────────────┘    │
│                    ↓                        │
│              第 N+1 回合                     │
└─────────────────────────────────────────────┘
```

### 我方回合规则
1. 玩家**自由选择**我方任意未行动单位进行操作
2. 每个单位每回合可执行：
   - **移动**：消耗MOV值在蓝色范围内移动
   - **主行动**：攻击/技能/使用物品/等待
   - 移动与主行动**顺序可调换**（先打后跑/先跑后打）
3. 确认后**不可撤销**（硬核战棋传统）
4. 全部单位行动完毕后，手动点击"结束回合"或自动触发（所有单位已行动）
5. 可提前结束回合（跳过剩余未行动单位）

### 敌方回合规则
1. AI控制所有敌方单位，按**SPD从高到低**依次行动
2. 每个敌方单位独立完成"移动+行动"后，轮到下一个
3. 全部敌方单位行动完毕后，自动切换回我方回合
4. 回合计数器+1

### SPD的作用
在传统回合制中，SPD不再驱动ATB速度条，而是：
- **我方回合**：决定右侧行动队列的默认排序（玩家可无视，自由选择）
- **敌方回合**：决定敌方单位的行动先后顺序
- **闪避率**：(SPD × 1.0)%，上限30%
- **双击判定**：若攻击方SPD ≥ 目标SPD + 5，攻击方追加一次攻击（伤害×50%）

### 凯恩的指挥技（回合干预）
- **战术指挥**：消耗1战术点，让我方一名已行动完毕的单位**恢复未行动状态**（每回合限1次）
- **紧急指令**：消耗2战术点，在敌方回合中**插入一次我方行动**
- **重整旗鼓**：消耗3战术点，跳过当前敌方回合的剩余单位行动

## 三、六维属性系统

### 属性定义

| 属性 | 全称 | 说明 | 典型范围(Lv1) |
|------|------|------|---------------|
| STR | Strength | 物理攻击力，影响物理技能伤害 | 8~18 |
| MAG | Magic | 魔法攻击力，影响魔法技能伤害 | 6~20 |
| SPD | Speed | 速度，决定敌方回合行动顺序、闪避率、双击判定 | 6~16 |
| DEF | Defense | 物理防御，减少物理伤害 | 6~18 |
| RES | Resistance | 魔法抗性，减少魔法伤害 | 6~18 |
| MOV | Movement | 移动力，每回合可移动格数 | 3~6 |

### 衍生属性
| 属性 | 计算公式 |
|------|----------|
| 最大HP | 基础HP + (等级×成长系数) + 装备加成 |
| 最大MP | 基础MP + (MAG×0.5) + (等级×成长系数) |
| 物理攻击力 | STR + 武器ATK |
| 魔法攻击力 | MAG + 武器MATK |
| 物理防御力 | DEF + 防具DEF |
| 魔法防御力 | RES + 防具RES |
| 命中率 | 基础90% + (SPD×0.5)% + 装备修正 |
| 闪避率 | (SPD×1.0)% + 装备修正（上限30%） |
| 暴击率 | 5% + (SPD×0.3)% + 装备修正 |
| 暴击伤害 | 150% + 装备修正 |

## 四、伤害计算公式

### 物理伤害
```
基础伤害 = (攻击方STR + 武器ATK) × 技能倍率
减伤率 = 防御方DEF / (防御方DEF + 50 + 攻击方等级×2)
最终伤害 = 基础伤害 × (1 - 减伤率) × 浮动系数(0.9~1.1)
暴击时：最终伤害 × 暴击伤害倍率
```

### 魔法伤害
```
基础伤害 = (攻击方MAG + 武器MATK) × 技能倍率
减伤率 = 防御方RES / (防御方RES + 50 + 攻击方等级×2)
最终伤害 = 基础伤害 × (1 - 减伤率) × 浮动系数(0.9~1.1)
```

### 地形修正
| 条件 | 伤害修正 |
|------|----------|
| 攻击方位于高地(+1 vs 目标) | +20% |
| 攻击方位于低地(-1 vs 目标) | -15% |
| 目标在掩体后（正面攻击） | -30% |
| 目标在草丛中 | 命中率-15% |
| 目标在水中 | 雷属性伤害+30% |

## 五、技能系统

### 技能分类
| 类型 | 说明 | 消耗 |
|------|------|------|
| 基础攻击 | 每个角色固有，无冷却 | 无消耗 |
| 主动技能 | 4个技能槽位，升级可获得/替换 | MP/AP/HP |
| 被动技能 | 常驻效果，部分有触发条件 | 无消耗 |
| 羁绊连携 | 相邻友方同时参与的特殊技能 | MP共享 |
| 指挥技能 | 仅凯恩可用，全局效果 | 战术点数 |

### 技能资源
- 技能消耗**MP**（魔力值），每回合自动回复MP×10%
- 无MP消耗的技能使用**冷却回合数**限制
- 基础攻击无消耗无冷却
- 部分装备可提升MP上限或回复速度

### 技能范围形状
| 形状 | 说明 | 示例技能 |
|------|------|----------|
| 单点 | 1格 | 基础攻击 |
| 直线 | N格直线 | 穿云箭 |
| 扇形 | 前方3格扇形 | 火焰斩 |
| 十字 | 十字4格 | 圣光爆 |
| 菱形 | 周围菱形 | 战吼光环 |

## 八、AI行为系统

### 行为树结构
```
Root
├── Selector（优先级选择）
│   ├── 生存判断
│   │   ├── HP < 30% → 撤退/使用治疗技能
│   │   └── 受到持续伤害 → 移动出危险区域
│   ├── 击杀判断
│   │   └── 可一击击杀友方 → 优先攻击该目标
│   ├── 技能判断
│   │   ├── AOE技能覆盖≥2目标 → 释放AOE
│   │   ├── 治疗技能有目标HP<50% → 治疗
│   │   └── 控制技能 → 优先控制高威胁目标
│   ├── 战术判断
│   │   ├── 可占据高地 → 向高地移动
│   │   └── 可破坏地形造成伤害 → 攻击地形
│   └── 默认攻击
│       └── 攻击范围内最近的目标
```

### AI难度分级
| 难度 | 行为特点 |
|------|----------|
| 普通 | 基础行为树，少量随机性 |
| 困难 | 完整行为树，考虑朝向/地形，会集火 |
| 噩梦 | 预判玩家行动，优先击杀治疗/脆皮，完美利用地形 |
