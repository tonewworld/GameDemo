# UI_Battle - 战斗界面设计

> [Demo] 实现核心HUD：棋盘、行动队列、信息面板、行动轮盘。小地图和战斗日志可选。

## 概述

战斗界面是游戏最核心的UI，承载战棋操作与信息展示。设计目标：信息密度高但不混乱，操作路径短。

## 区域详解

### 1. 顶部信息栏
- 左侧：当前层数、波次、章节名
- 右侧：金币、灵魂碎片数量
- 点击灵魂碎片可查看本局已获得的所有奖励记录

### 2. 棋盘区域（主体）
- 2.5D斜俯视角，网格可切换显示/隐藏（快捷键G）
- 移动范围：蓝色高亮格子
- 攻击范围：红色高亮格子
- 技能范围：橙色高亮格子（形状因技能而异：直线/扇形/十字/菱形）
- 当前操作角色脚下有光圈指示
- 悬停敌人时显示伤害预览面板

### 3. 行动队列面板（左侧竖条）
```
┌──────────────────┐
│  ★ 我方回合 ★    │
│ ─────────────── │
│ ✓ 艾拉    ◄     │  ← 当前操作（已行动，可再选）
│ ○ 凯恩          │  ← 未行动
│ ○ 莉莉丝        │  ← 未行动
│ ○ 铁砧          │  ← 未行动
│ ─────────────── │
│  [结束回合 →]   │  ← 全部行动完毕后点击
│                  │
│  敌方回合等待中... │
└──────────────────┘
```
- ★ = 当前选中角色，✓ = 已行动完毕，○ = 尚未行动
- 点击任意未行动角色头像切换操作目标
- "结束回合"按钮在我方回合始终可见
- 敌方回合时面板显示敌方行动进度（当前行动的敌人高亮）
- 头像旁显示SPD数值作为行动顺序参考
- 点击头像可聚焦镜头到该角色

### 4. 小地图（右上角）
- 棋盘缩略图，绿色=友方，红色=敌人，黄色=中立
- 相机视锥范围用白框标识
- 点击小地图可快速移动相机

### 5. 选中角色信息面板（底部左侧）
```
┌──────────────────────────────────────────────┐
│ [头像] 艾拉  Lv.8  银骑士                    │
│ HP ████████████████░░░░░  85/100             │
│ MP ██████████░░░░░░░░░░░  50/65              │
│ ─────────────────────────────────────────   │
│ STR:15  DEF:18  SPD:12  MAG:8  RES:14       │
│ MOV ████░░  4/5   状态：[铁壁]              │
│ 装备：银枪 | 骑士盾 | 生命护符               │
└──────────────────────────────────────────────┘
```
- HP条颜色：绿(>60%) → 黄(30-60%) → 红(<30%)
- 状态图标显示在角色头像右下角，悬停查看详情

### 6. 行动轮盘（底部中央，当前角色回合时弹出）
- 扇形或横向排列6个选项
- 4个主动技 + 物品 + 等待
- 每个技能显示消耗（AP/MP/HP）
- 可用技能高亮，不可用技能置灰并显示原因（"MP不足"/"冷却中：2回合"）
- 选中技能后进入目标选择模式

### 7. 回合信息（右侧面板）
- 当前回合数 + 当前阶段标识（"我方回合"绿色 / "敌方回合"红色）
- 我方剩余可行动单位数
- 剩余敌人数（已击杀/总数）
- 任务目标文本
- 逃离条件（若有）
- 回合自动结束倒计时（4秒无操作自动切换，可关闭）

## 子界面

### 伤害预览面板
悬停目标时在目标头顶弹出：
```
┌──────────────────────┐
│ 艾拉 → 恶魔士兵       │
│ ──────────────────  │
│ 命中率：  92%        │
│ 预伤：   38 (25~42)  │
│ 暴击率：  15% (侧面) │
│ ──────────────────  │
│ 击杀：   需要2次攻击  │
│ 反击伤： 12 (不可击杀)│
└──────────────────────┘
```
- 预伤范围显示最小-最大（考虑随机浮动和暴击）
- 反击信息在敌人有反击能力时显示
- 支持快捷键确认（右键/空格）快速执行

### 技能详情弹窗
选中技能时弹出：
```
┌──────────────────────────────┐
│ 圣光斩  [MP:12] [冷却:2回合]  │
│ ──────────────────────────  │
│ 类型：物理/神圣               │
│ 范围：前方直线3格             │
│ 伤害：STR × 1.5 + 10        │
│ 附加：对恶魔系伤害+30%       │
│ 消耗：MP 12，冷却：2回合     │
│ ──────────────────────────  │
│ "以神圣之力灌注剑刃"         │
└──────────────────────────────┘
```

## 镜头控制

| 操作 | 键鼠 | 手柄 |
|------|------|------|
| 平移 | WASD / 鼠标边缘 | 左摇杆 |
| 旋转 | Q/E | LB/RB |
| 缩放 | 滚轮 | RT/LT |
| 聚焦角色 | Tab循环 / 点击头像 | Y |
| 重置视角 | Home | 右摇杆按下 |
| 网格开关 | G | Select键 |
