# Module_Event - 事件系统（Demo版）

> [Demo] 3个事件模板，不做属性检定、角色挑战、赌博。

---

## Demo 事件模板

### 事件1：废墟中的平民
```
"一个平民被困在坍塌的石柱下，正在绝望呼救。"

选项A：[救援] 消耗一瓶治疗药水 → 平民获救，获得稀有装备×1
选项B：[离开] → 无事发生
```

### 事件2：受伤的使魔
```
"一只受伤的使魔蜷缩在角落，它的翅膀在流血。"

选项A：[治疗] 消耗一瓶治疗药水 → 使魔感激，获得灵魂碎片×100
选项B：[离开] → 无事发生
选项C：[捕捉] → 获得临时BUFF"使魔的祝福"(下3场战斗MAG+5)
```

### 事件3：古老的弈者笔记
```
"地上散落着一位前辈弈者的笔记残页。"

选项A：[研读] → 英雄获得额外经验×80
选项B：[带走] → 获得消耗品"灵魂水晶"×1
选项C：[焚烧] → 获得金币×50
```

> [Demo] 3个事件随机出现，选项不需要属性检定（简化开发）。事件结果直接影响队伍状态/物品/金币。
